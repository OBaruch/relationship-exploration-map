Mapa Relacional Interactivo

Una herramienta visual tipo juego para explicar apego, exploración, límites, minas sensibles, reciprocidad y madurez relacional. El tablero está espejado: si una persona abre un bloque, el equivalente del otro lado queda disponible. Si una persona coloca una mina, también aparece del otro lado como regla recíproca.

Centro = base segura Espejo = reciprocidad Bloques = experiencias Minas = no negociables / riesgos Desactivar = trabajo consciente

Cómo usarlo

  • Modo Editar minas: define juntos qué cosas son sensibles o no negociables.
  • Modo Explorar: vayan desbloqueando bloques de la relación.
  • Modo Completar: marquen experiencias ya vividas o integradas.
  • Modo Transformar: conviertan una mina trabajada en un bloque resignificado.

Abajo encontrarás tutoriales de apego y ejemplos de tableros maduros para explicar distintas dinámicas.

Izquierda = Persona A Derecha = Persona B

Lectura conceptual

Centro: el vínculo base, seguridad, pertenencia y regreso.

Fila superior: juego, viajes, novedad, amistades y expansión.

Fila media: confianza, rutina, familia, espacio personal, coordinación.

Fila inferior: sexualidad, celos, secretos, control, traición y no negociables.

El espejo hace visible una idea central: si algo aplica para uno, aplica para ambos.

Tutorial visual de tipos de apego

Apego seguro

Explora sin sentir que se pierde la relación. Sale, vuelve, tolera distancia y novedad.

Apego ansioso

El otro ya abrió espacio, pero esta persona se queda cerca del centro buscando seguridad.

Apego evitativo

Se mueve, pero evita los bloques que implican mucha intimidad o dependencia emocional.

Apego desorganizado

Va y viene. Se acerca, huye, repite riesgos y le cuesta sostener dirección estable.
