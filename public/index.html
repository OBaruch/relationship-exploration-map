Mapa Relacional Interactivo

Herramienta visual para conversar sobre apego, limites, exploracion, reciprocidad y reparacion. El tablero esta espejado: cada accion de un lado impacta el bloque equivalente del otro lado.

Centro = base segura Espejo = contrato reciproco Minas = limites sensibles Transformar = trabajo consciente Reflexionar = tema conversado
